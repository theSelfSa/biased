# @biased/web

Next.js App Router web experience for B.I.A.S.E.D.

## Responsibilities

- marketing landing page
- Better Auth session handling
- business workspace creation flow
- dashboard, imports, documents, action center, and AI investigation surfaces
- PWA shell for Android and Windows reach

## Local commands

- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm auth:migrate`
- `pnpm db:migrate`
